Chibi Animé — Claude

Rigging & animation par IA

Un personnage chibi modélisé sur Blender dont l'armature Mixamo avait été perdue. Avec Claude Code, on a reconstruit le squelette de 25 os en analysant les vertex groups existants et en calculant la position de chaque os par moyenne pondérée des vertices.

Le cycle de marche a été entièrement généré en Python — 8 poses clés sur 30 frames, avec rebond vertical et modificateurs cycliques pour une boucle infinie. Aucune intervention manuelle dans Blender : tout a été piloté depuis le terminal.

Blender · Python · Claude Code · EEVEE
Animate 3D — Motion Capture

De la vidéo à l'animation 3D

Animate 3D est une app qui transforme une vidéo de référence en animation squelettique importable dans Blender. Le pipeline passe par trois étapes :

Vidéo → MediaPipe (2D) → MotionBERT (3D) → BVH

MediaPipe extrait les poses 2D frame par frame, MotionBERT les convertit en mouvements 3D, et le résultat est exporté en BVH — un format standard d'animation squelettique. C'est ce pipeline qui a généré l'armature de 17 os et les 139 frames de données de mouvement utilisées sur le projet chibi ci-dessus.
